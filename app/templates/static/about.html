Щербина Ярослав

Full-Stack Developer · Flask · Python · JS

VideoHub — это мой личный проект, созданный с нуля с целью доказать, что качественная видеоплатформа не требует огромной корпоративной инфраструктуры. Каждая строчка кода написана с вниманием к деталям: от безопасного хранения паролей до YouTube-подобного алгоритма рекомендаций.

Стек: Python · Flask · SQLAlchemy · ffmpeg · Pillow на бэкенде, чистый HTML5 · CSS Variables · Vanilla JS на фронтенде. База данных — PostgreSQL в продакшне, SQLite для разработки. Аутентификация — JWT для API, Flask-Login для сессий.
